JDBC Setup

JDBC is the Java database interface to work with all of the major databases. This information is used to connect to the database, and defines the type of database.

From the OABuilder menu, select "View" and then select "DataSource Settings ...".


This allows you to create a new database/datasource type that can then be used by your model. The only required data is the "Database Type", so that the code generators can "know" the type of program to create.
