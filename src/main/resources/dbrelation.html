Relationships

OABuilder allows for specific information when creating a Link/Relationship between two objects. This is dependend on the link type, as follows:
  • One-One: it will create foreign keys in each table.
  • One-Many: a foreign key will be created in the Many table that will map to the primary key column in the One table.
  • Many-Many: a link table is created that will be used to create the relationship between the objects.

    • The "Editing a Link" section has additonal information.
