Callbacks

Callbacks are used in Objects to allow programs to interact with Objects.

A callback is implemented by creating a callback method in the Object java code, or as a callback property method.

OAObjCallback is the annotation used to mark and Object method as a callback.

OAObjectCallback is the paramater passed and updated.
Object callbacks.

Example: 

   
public void callback(final OAObjectCallback callback) {
    if (callback == null) return;
    switch (callback.getType()) {
    case AllowNew:
        // todo: custom code here;
        callback.ack();
        // callback.setAllowed(true); 
        // callback.setResponse("");
        break;
    }
}

Property callbacks. 

public void inspectedDateCallback(final OAObjectCallback callback) {
    if (callback == null) return;
    switch (callback.getType()) {
    case AllowSave:
        // todo: custom code here;
        callback.ack();
        callback.setAllowed(true); 
        callback.setResponse("");
        break;
    }
}
