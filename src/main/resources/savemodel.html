Saving a Model

OABuilder will save an OAModel to a file with the default suffix ".obx". The file is created as a text file using XML formatting.

To save the Model, choose one of the following:

  • "save" command from the toolbar
  • "Save Model" from the "File" menu.
  • "Save Model As ..." from the "File" menu.
There are also commands to save as an image, and Pdf file.
As with any software, it is advised to frequently save your work and to also have backup copies created on a regular basis. OABuilder currently keeps a copy of each saved file in a backup directory, located where OABuilder is running.
