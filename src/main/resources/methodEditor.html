Method Editor

Methods are additional functions added to an Object.


General information.





Source code for methods. There are options to also have the code executed on the server.






Method Editor.



General tab.

  • Name
  • Upper Name
  • Display Name
  • Description
  • Method Signature
  • Tool Tip
  • Attach to - proeprty that this command can be attached to.
  • Command button - to create as button.
  • Password Property - requires user to enter before running command.
  • Confirmation Message
  • Show Processing
  • Console Property - property that is flagged as "console", that will be displayed while command is running.
  • Completed Message - displayed when command is finished.
  • Run on server - have the method run on the server (remoted).
  • Supports Multiple at Once - allow selecting more than one (collection) to run command on.



Input Properties tab. Creates a Request object for entering information needed.

  • Add - use existing property.
  • New - create a new property.



Code tab.

  • Generate Code - creates template for method code.



Code tab, Callback subtab.

  • Generate Code - creates a template for adding custom Callback code.
