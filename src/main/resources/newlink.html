Creating Links/Relationships between Objects

    Using the Link command
  1. Select the Link command button on the toolbar or select "Create OALink" from the "Edit" menu.
  2. Click on the first object to line, holding down the mouse button and dragging to the other object.
  3. Release mouse button when the second object is selected, this will create the link.


    Using right-click menu to create new link
  1. Right click the object to link.
  2. From the popup menu, select "create new link". This will display the "create new link" dialog.
  3. Select the object to link to.
