Opening an existing Model

To open an existing OA Model, choose the model name from the "file" menu, or select "open" from the "file" menu to select from file manager. There is also an open button on the toolbar.

If opening a model using the file manager, you will need to select the model file from the disk location, with the file name ending in ".obx". This is an XML file structure that OABuilder uses to store models.
